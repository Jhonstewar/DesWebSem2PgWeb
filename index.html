Comparativa de Servicios de Hosting

1. Vercel

Comenzando con Vercel se encontró que es un hosting enfocado en aplicaciones web desarrolladas con tecnologías como JavaScript como react o Vue. Vercel ofrece un flujo de despliegue automatizado desde varios sistemas de versionamientos como github o bitbucket que despliegan inmediatamente luego de cualquier cambio. Soporta gunciones como serverless, https gratuito y permite configuraciónes avanzadas desde el plan gratuito. En su plan gratuito permite proyectos ilimitados.

2. Netlify

En Netlify también es una plataforma popular en el despliegue de sitios estáticos y JAMstack. Al igual que Vercel se conecta fácilmente con los distintos repositorios y permite automatizar desplieges. Ofrece funciones como formularios, autentificación y serverless. El plan gratuito incluye https, ci/cd y despliegue de sitios desde diferentes ramas.

3. GitHub Pages

Y por ultimo en GitHub Pages es un servicio ofrecido directamente por GitHub que permite publicar páginas estáticas directamente desde un repositorio. Es ideal para proyectos sencillos que no requieren funciones dinámicas ni servidores backend.

Pasos para subir la aplicación web

  1. Crear un repositorio en GitHub con nombre, por ejemplo, pagina-tarea.
  2. Clonar el repositorio en tu PC.
  3. Copiar los archivos index.html, style.css y script.js.
  4. Hacer git add ., git commit -m "Subiendo tarea" y git push.
  5. Desde Vercel.com, iniciar sesión con GitHub y seleccionar el repositorio para desplegar.
  6. Esperar que compile y se muestre la URL del proyecto.
