📊 Comparación de proveedores de hosting

GitHub Pages: Ideal para proyectos estáticos. Es gratuito, permite conectar con GitHub directamente y tiene una comunidad grande.

Netlify: Ofrece despliegue automático desde GitHub, funciones serverless y dominio gratuito. Es muy usado en JAMstack.

Vercel: Excelente para React y Next.js. Despliegue automático desde repositorios y preview por cada PR. Gratuito con funciones premium.
